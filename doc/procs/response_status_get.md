qc::reponse status get
===========

part of [Global JSON Response API](../response_api.md)

Usage
-----
`qc::reponse status get`

Description
-----------
Get the status for the JSON Response.

Examples
--------
```tcl

% qc::reponse status valid
status valid

% qc::reponse status get
valid

% qc::reponse status invalid
status invalid

% qc::reponse status get
invalid

```
