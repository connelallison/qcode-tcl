qc::margin
==========

part of [Docs](../index.md)

Usage
-----
`
        qc::margin cost price ?dec_places?
    `

Description
-----------
Calculates the gross margin on supplied cost and revenue

Examples
--------
```tcl

% qc::margin 0.40 2.99
86.6
% qc::margin 0.40 2.99 3
86.622
% qc::margin 0.40 0.40
0.0
```
