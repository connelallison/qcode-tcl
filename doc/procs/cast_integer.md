qc::cast_integer
================

part of [Casting Procs](../cast.md)

Usage
-----
`qc::cast_integer string`

Description
-----------
Deprecated - see [qc::cast integer][1]
Try to cast given string into an integer

Examples
--------
```tcl

% cast_integer 2,305
% 2305
% 
% cast_integer 2.366%
% 2
%
% cast_integer 43e2
4300

```
