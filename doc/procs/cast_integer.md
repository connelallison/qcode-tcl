qc::cast_integer
================

part of [Casting Procs](../qc/wiki/CastPage)

Usage
-----
`qc::cast_integer string`

Description
-----------
Try to cast given string into an integer

Examples
--------
```tcl

% cast_integer 2,305
% 2305
% 
% cast_integer 2.366%
% 2
%
% cast_integer 43e2
4300

```
