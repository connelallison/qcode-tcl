qc::date_iso_week_end
=====================

part of [Docs](../index.md)

Usage
-----
`qc::date_iso_week_end date`

Description
-----------
Returns the date of the end of the week in which $date falls

Examples
-----------
```
% date_iso_week 2007-05-06
% 18
%
% date_iso_week "today"
% 32
```
