qc::format_date
===============

part of [Date Handling](../qc/wiki/DateHandling)

Usage
-----
`qc::format_date date`

Description
-----------
Format a date for the user<br/>Will be customizable in future but at present chooses the ISO format.

Examples
--------
```tcl

% format_date &quot;23rd June 2008&quot;
% 2008-06-23
%
% format_date 2/5/08
% 2008-05-02

```
