qc::max_nz
==========

part of [Docs](.)

Usage
-----
`
        qc::max_nz val1 ?val2? ?val3? ...
    `

Description
-----------
Return the maximum supplied value which is non zero

Examples
--------
```tcl

% qc::max_nz 0 1 5 7 3
7
% qc::max_nz 0 0 0 0 0
% 
```
