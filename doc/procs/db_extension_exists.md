qc::db_extension_exists
===========

part of [Database API](../db.md)

Usage
-----
`db_extension_exists extension_name`

Description
-----------
Checks if the given extension exists in the database running on pool "DEFAULT".

Examples
--------
```tcl

% db_extension_exists pgcrypto
true

```
