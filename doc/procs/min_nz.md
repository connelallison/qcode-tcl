qc::min_nz
==========

part of [Docs](../index.md)

Usage
-----
`
        qc::min_nz val1 ?val2? ?val3? ...
    `

Description
-----------
Return the minimum supplied value which is non zero

Examples
--------
```tcl

% qc::min_nz 0 1 5 7 3
1
```
