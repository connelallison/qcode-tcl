qc::date_day_shortname
======================

part of [Date Handling](../date.md)

Usage
-----
`qc::date_day_shortname date`

Description
-----------
Return the day of the week as Mon,Tue,Wed,Thu,Fri,Sat,Sun

Examples
--------
```tcl

% date_day_shortname 2007-05-06
% Sun
%
% date_day_shortname "2 days ago"
% Mon

```
