qc::auth
========

part of [Authentication](../qc/wiki/AuthPage)

Usage
-----
`qc::auth `

Description
-----------
Try to authenticate the current employee/user<br/>If successful cache the result in global<br/>On failure throw AUTH error

Examples
--------
```tcl

% set employee_id [auth]
23

```
