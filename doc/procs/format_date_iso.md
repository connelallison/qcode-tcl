qc::format_date_iso
===================

part of [Date Handling](../qc/wiki/DateHandling)

Usage
-----
`qc::format_date_iso date`

Description
-----------
Format a date as an ISO 8601 string like 2006-04-28

Examples
--------
```tcl

% format_date &quot;23rd June 2008&quot;
% 2008-06-23
%
% format_date 2/5/08
% 2008-05-02

```
