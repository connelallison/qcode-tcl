qc::split_pair
==============

part of [Docs](.)

Usage
-----
`
        qc::split_pair string delimiter 
    `

Description
-----------
Split a string into 2 parts at the first occurence of the delimiter

Examples
--------
```tcl

% qc::split_pair &quot;key=key_value&quot; =
key key_value
```
