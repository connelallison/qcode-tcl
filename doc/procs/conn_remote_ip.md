qc::conn_remote_ip
==================

part of [Docs](../index.md)

Usage
-----
`qc::conn_remote_ip `

Description
-----------
Try to return the remote IP address of the current connection<br/>Trust that a reverse proxy like nginx is setup to pass an X-Forwarded-For header.

Examples
--------
```tcl

% conn_remote_ip
12.34.56.78

```
