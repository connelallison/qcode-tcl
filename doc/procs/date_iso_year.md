qc::date_iso_year
=================

part of [Date Handling](../qc/wiki/DateHandling)

Usage
-----
`qc::date_iso_year date`

Description
-----------
Return the ISO year.

Examples
--------
```tcl

% date_iso_year 2007-05-06
% 2007

```
