qc::format_cc_bin
=================

part of [Docs](.)

Usage
-----
`qc::format_cc_bin cc_no ?prefix?`

Description
-----------
First prefix digits of cc_no

Examples
--------
```tcl

% format_cc_bin 4111111111111111
411111
% format_cc_bin &quot;4111 **** **** 1111&quot; 4
4111

```
