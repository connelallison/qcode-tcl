qc::dates
=========

part of [Date Handling](../qc/wiki/DateHandling)

Usage
-----
`qc::dates from_date to_date`

Description
-----------
Return a list of dates from $from_date to $to_date inclusive.

Examples
--------
```tcl

% date_days 2007-02-25 2007-03-05
% 8
% date_days 2007-02-25 2007-02-26
% 1

```
