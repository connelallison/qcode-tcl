qc::pkcs_padding_strip
======================

part of [Docs](.)

Usage
-----
`qc::pkcs_padding_strip string`

Description
-----------
Trim PKCS padding from end of string.

Examples
--------
```tcl

% set pkcs_padding_append &quot;Hello World\5\5\5\5\5&quot;
Hello World

```
