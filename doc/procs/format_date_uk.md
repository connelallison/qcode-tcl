qc::format_date_uk
==================

part of [Date Handling](../qc/wiki/DateHandling)

Usage
-----
`qc::format_date_uk date`

Description
-----------
Format a date in UK format e.g. 27/03/07

Examples
--------
```tcl

% format_date_uk 2008-06-23
23/06/08
% 
% format_date_uk today
17/10/07

```
