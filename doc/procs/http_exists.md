qc::http_exists
===============

part of [Docs](.)

Usage
-----
`qc::http_exists args`

Description
-----------
Test if an URL returns a valid response

Examples
--------
```tcl

% qc::http_exists www.qcode.co.uk
true

% qc::http_exists www.qcode.co.uk/foo
false

```
