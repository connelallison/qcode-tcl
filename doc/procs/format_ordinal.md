qc::format_ordinal
==================

part of [Docs](../index.md)

Usage
-----
`qc::format_ordinal number`

Description
-----------
Format number with suffix 23 -> 23rd or 4 -> 4th

Examples
--------
```tcl

% format_ordinal 23
23rd
% format_ordinal 4
4th

```
