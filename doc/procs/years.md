qc::years
=========

part of [Date Handling](../qc/wiki/DateHandling)

Usage
-----
`qc::years from_date to_date`

Description
-----------
Returns list of years between from_date & to_date

Examples
--------
```tcl

% years 2007-02-25 2009-03-05
% 2007 2008 2009
% years 2007-02-25 2007-02-26
% 2007

```
