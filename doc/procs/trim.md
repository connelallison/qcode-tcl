qc::trim
========

part of [Docs](.)

Usage
-----
`
        qc::trim string
    `

Description
-----------
Removes and leading or trailing white space.

Examples
--------
```tcl

% set string &quot;       Testing          &quot;
    Testing          
% qc::trim $string
Testing
```
