qc::date_iso_week_start
=======================

part of [Docs](../index.md)

Usage
-----
`qc::date_iso_week_start date`

Description
-----------
Returns the date of the start of the week in which $date falls

Examples
-----------
```
% date_iso_week_start 2007-05-06
% 2007-04-30
%
% date_iso_week_start "today"
% 2012-08-06
```
