qc::format_timestamp_http
=========================

part of [Docs](.)

Usage
-----
`qc::format_timestamp_http string`

Description
-----------
Format string as http timestamp according to RFC 1123

Examples
--------
```tcl

% format_timestamp_http now
    Tue, 12 Jun 2012 10:39:47 GMT

```
