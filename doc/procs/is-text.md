qc::is text
==============

part of [Docs](../index.md)

Usage
-----
`qc::is text string`

Description
-----------
Checks if the given string is text.

Examples
--------
```tcl

% qc::is text "Hello World"
1
% qc::is text 12345
1
% qc::is text "" 
1
```
