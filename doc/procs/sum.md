qc::sum
=======

part of [Docs](../index.md)

Usage
-----
`
        qc::sum n1 ?n2? ?n3? ....
    `

Description
-----------
Returns the sum of list of numbers.

Examples
--------
```tcl

% qc::sum 1
1
% qc::sum 1 1 1 1 1 1 1 1 
8
```
