qc::date_year_start
===================

part of [Date Handling](../qc/wiki/DateHandling)

Usage
-----
`qc::date_year_start date`

Description
-----------
Return the date on the first day of the year

Examples
--------
```tcl

% date_year_start 2007-05-06
% 2007-01-01
%
% date_year_start "last year"
% 2006-01-01

```
