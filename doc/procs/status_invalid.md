qc::status invalid
===========

part of [Global JSON Response API](../response_api.md)

Usage
-----
`qc::status invalid`

Description
-----------
Set the status as "invalid" for the JSON Response.

Examples
--------
```tcl

% qc::status invalid
status invalid

```
