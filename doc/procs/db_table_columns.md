qc::db_table_columns
===========

part of [Database API](../db.md)

Usage
-----
`db_table_columns table`

Description
-----------
Returns a list of columns for the given table.

Examples
--------
```tcl

% db_table_columns users
user_id firstname surname email password_hash user_state user_image_url ip

```
