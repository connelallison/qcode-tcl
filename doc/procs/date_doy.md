qc::date_doy
============

part of [Date Handling](../qc/wiki/DateHandling)

Usage
-----
`qc::date_doy date`

Description
-----------
Return the day of year number.

Examples
--------
```tcl

% date_doy 2009-01-01
% 1
%
% date_doy 2009-12-31
% 365

```
