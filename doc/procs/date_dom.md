qc::date_dom
============

part of [Date Handling](../qc/wiki/DateHandling)

Usage
-----
`qc::date_dom date`

Description
-----------
Return the day of the month.

Examples
--------
```tcl

% date_dom 2007-05-06
% 6
%
% date_dom "2 days ago"
% 4

```
