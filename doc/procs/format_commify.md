qc::format_commify
==================

part of [Docs](.)

Usage
-----
`qc::format_commify number`

Description
-----------
Commify number into groups of three 2314 -> 2,314

Examples
--------
```tcl

% format_commify 2314
2,314
%
% format_commify 865425833.2354
865,425,833.2354

```
