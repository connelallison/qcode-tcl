qc::is_timestamp_http
=====================

part of [Docs](.)

Usage
-----
`qc::is_timestamp_http date`

Description
-----------
Returns true if date is an acceptable HTTP timestamp. Note although all three should be accepted,<br/>only RFC 1123 format should be generated.
