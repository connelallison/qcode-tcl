qc::date_month_shortname
========================

part of [Date Handling](../date.md)

Usage
-----
`qc::date_month_shortname date`

Description
-----------
Return the short month name Jan,Feb,Mar etc.

Examples
--------
```tcl

% date_month_shortname 2007-08-06
% Aug
%
% date_month_shortname "2 days ago"
% Apr

```
