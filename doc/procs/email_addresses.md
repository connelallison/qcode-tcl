qc::email_addresses
===================

part of [Sending Email](../email.md)

Usage
-----
`qc::email_addresses text`

Description
-----------
Return a list of email addresses in the text

Examples
--------
```tcl

% qc::email_addresses {"Joe Biden" <joe@biden.com> "Paul Ryan" <paul@ryan.com>}
joe@biden.com paul@ryan.com

```
