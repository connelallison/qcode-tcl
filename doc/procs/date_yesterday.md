qc::date_yesterday
==================

part of [Docs](../index.md)

Usage
-----
`qc::date_yesterday date`

Description
-----------
Return yesterday's date

Examples
-----------
```
% date_yesterday 2007-05-06
% 2007-05-05
%
% date_yesterday "today"
% 2012-08-09
```
