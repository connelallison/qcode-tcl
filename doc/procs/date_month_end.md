qc::date_month_end
==================

part of [Date Handling](../date.md)

Usage
-----
`qc::date_month_end date`

Description
-----------
Return the date on the last day of the month for the date given

Examples
--------
```tcl

% date_month_end 2007-05-06
% 2007-05-31
%
% date_month_end today
% 2008-02-29

```
