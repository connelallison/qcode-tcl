qc::exp2string
==============

part of [Docs](.)

Usage
-----
`
        qc::exp2string number 
    `

Description
-----------
Convert floats including exponentials to strings

Examples
--------
```tcl

% qc::exp2string -1.23e6
-1230000
% qc::exp2string 10000
10000
```
