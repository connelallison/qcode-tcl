qc::base
========

part of [Docs](.)

Usage
-----
`
        qc::base base number
    `

Description
-----------
Convert supplied number to specified base

Examples
--------
```tcl

% qc::base 2 1024
10000000000
% qc::base 8 1024 
2000
% qc::base 16 15
F
```
