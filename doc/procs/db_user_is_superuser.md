qc::db_user_is_superuser
===========

part of [Database API](../db.md)

Usage
-----
`db_user_is_superuser username`

Description
-----------
Checks if the given user is a superuser for the database running on pool "DEFAULT".

Examples
--------
```tcl

% db_user_is_superuser testuser
true

```
