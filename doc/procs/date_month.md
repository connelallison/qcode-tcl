qc::date_month
==============

part of [Date Handling](../qc/wiki/DateHandling)

Usage
-----
`qc::date_month date`

Description
-----------
Return the month number.

Examples
--------
```tcl

% date_month 2007-05-06
% 5
%
% date_month "last month"
% 4

```
