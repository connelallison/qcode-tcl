qc::db_enum_values
===========

part of [Database API](../db.md)

Usage
-----
`db_enum_values enum_name`

Description
-----------
Returns a list of the values for the given enumeration.

Examples
--------
```tcl

% db_enum_values user_state
ACTIVE DISABLED

```
