qc::date_quarter
================

part of [Date Handling](../qc/wiki/DateHandling)

Usage
-----
`qc::date_quarter date`

Description
-----------
Return the quarter the date is in

Examples
--------
```tcl

% date_quarter 2007-08-06
% 3

```
