qc::upper
=========

part of [Docs](.)

Usage
-----
`
        qc::upper string
    `

Description
-----------
Convert string to upper case

Examples
--------
```tcl

% qc::upper &quot;This will be changed to upper case.&quot;
THIS WILL BE CHANGED TO UPPER CASE.
```
