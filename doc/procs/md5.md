qc::md5
=======

part of [Docs](.)

Usage
-----
`
        qc::md5 string
    `

Description
-----------
Returns the md5 hash of supplied string.

Examples
--------
```tcl

1&gt; qc::md5 {This string requires hashing}
fed9e24fe3df8ca8c093fca78e546ddc
```
