qc::date_day_name
=================

part of [Date Handling](../qc/wiki/DateHandling)

Usage
-----
`qc::date_day_name date`

Description
-----------
Return the full day of the week as Monday,Tuesday etc

Examples
--------
```tcl

% date_day_name 2007-05-06
% Sunday
%
% date_day_name "2 days ago"
% Monday

```
