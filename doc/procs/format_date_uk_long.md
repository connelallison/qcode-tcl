qc::format_date_uk_long
=======================

part of [Date Handling](../qc/wiki/DateHandling)

Usage
-----
`qc::format_date_uk_long date`

Description
-----------
Format a date in UK format with a 4 digit year e.g. 27/03/2007

Examples
--------
```tcl

% format_date_uk 2008-06-23
23/06/2008
% 
% format_date_uk today
17/10/2007

```
