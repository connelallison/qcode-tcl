qc::lower
=========

part of [Docs](.)

Usage
-----
`
        qc::lower string
    `

Description
-----------
Convert string to upper case

Examples
--------
```tcl

% qc::lower "THIS WILL BE CHANGED TO LOWER"
this will be changed to lower
```
