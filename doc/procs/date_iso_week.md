qc::date_iso_week
=================

part of [Docs](../index.md)

Usage
-----
`qc::date_iso_week date`

Description
-----------
Return the ISO week number.

Example
-----------
```
% date_iso_week 2007-05-06
% 18
%
% date_iso_week "today"
% 32
```
