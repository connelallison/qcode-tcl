qc::file_alias_path_exists
===============

part of [Docs](../index.md)

Usage
-----
`file_alias_path_exists path`

Description
-----------
Check if this url path is an alias for a file.

Examples
--------
```tcl

% qc::file_alias_path_exists /foo
true

% qc::file_alias_path_exists /bar
false

```
