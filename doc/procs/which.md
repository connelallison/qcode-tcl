qc::which
=========

part of [Docs](../index.md)

Usage
-----
`
        qc::which command
    `

Description
-----------
Return path of unix command - cache result in nsv on AOLserver if present

Examples
--------
```tcl

% qc::which sftp
/usr/bin/sftp
```
