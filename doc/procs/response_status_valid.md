qc::response status valid
===========

part of [Connection Response API](../response_api.md)

Usage
-----
`qc::reponse status valid`

Description
-----------
Set the status as "valid" for the [connection response](../connection-response.md).

Examples
--------
```tcl

% qc::response status valid
status valid

```
