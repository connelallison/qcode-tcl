qc::dict_zipper
===========

part of [Docs](../index.md)

Usage
-----
`dict_zipper keys values`

Description
-----------
Creates a dictionary by pairing up each key and value from the provided lists.

Examples
--------
```tcl

% dict_zipper [list 1 2 3] [list foo bar baz]
1 foo 2 bar 3 baz

```
