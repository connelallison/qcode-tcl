qc::date_dow
============

part of [Date Handling](../date.md)

Usage
-----
`qc::date_dow date`

Description
-----------
Return the day of the week.

Examples
--------
```tcl

% date_dow 2007-05-06
% 7
%
% date_dow "2 days ago"
% 3

```
