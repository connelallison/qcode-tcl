qc::date_dow
============

part of [Date Handling](../qc/wiki/DateHandling)

Usage
-----
`qc::date_dow date`

Description
-----------
Return the day of the week.

Examples
--------
```tcl

% date_dow 2007-05-06
% 7
%
% date_dow &quot;2 days ago&quot;
% 3

```
