qc::is boolean
==============

part of [Docs](../index.md)

Usage
-----
`qc::is boolean string`

Description
-----------
Checks if the given string is a boolean.

Examples
--------
```tcl

% qc::is boolean 1
1
% qc::is boolean foo
0
% qc::is boolean true
1
% qc::is boolean no
1
```
