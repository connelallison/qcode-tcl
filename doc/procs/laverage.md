qc::laverage
============

part of [Docs](../index.md)

Usage
-----
`
        qc::laverage list
    `

Description
-----------
Returns average of all list elements

Examples
--------
```tcl

% set items [list 1 2 1 100]
1 2 1 100
% qc::laverage $items
26.0
```
