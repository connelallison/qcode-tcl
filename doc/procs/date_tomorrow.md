qc::date_tomorrow
=================

part of [Docs](../index.md)

Usage
-----
`qc::date_tomorrow date`

Description
-----------
Return tomorrow's date

Example
-----------
```
% date_tomorrow 2007-05-06
% 2007-05-07
%
% date_tomorrow "today"
% 2012-08-11
```
