qc::date_year_end
=================

part of [Date Handling](../qc/wiki/DateHandling)

Usage
-----
`qc::date_year_end date`

Description
-----------
Return the date on the last day of the year

Examples
--------
```tcl

% date_year_end 2007-05-06
% 2007-12-31
%
% date_year_end &quot;last year&quot;
% 2006-12-31

```
