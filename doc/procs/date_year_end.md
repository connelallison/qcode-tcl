qc::date_year_end
=================

part of [Date Handling](../date.md)

Usage
-----
`qc::date_year_end date`

Description
-----------
Return the date on the last day of the year

Examples
--------
```tcl

% date_year_end 2007-05-06
% 2007-12-31
%
% date_year_end "last year"
% 2006-12-31

```
