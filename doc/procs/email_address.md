qc::email_address
=================

part of [Sending Email](../qc/wiki/SendingEmail)

Usage
-----
`qc::email_address text`

Description
-----------
Extract the email address from text

Examples
--------
```tcl

    % qc::email_address {"Joe Biden" <joe@biden.com>}
joe@biden.com
```
