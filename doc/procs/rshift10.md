qc::rshift10
============

part of [Docs](.)

Usage
-----
`
        qc::rshift10 int dec_places
    `

Description
-----------
Move decimal point $dec_places to the left

Examples
--------
```tcl

% qc::rshift10 6 3
0.006
% qc::rshift10 23 1
2.3
```
