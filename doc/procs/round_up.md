qc::round_up
============

part of [Docs](.)

Usage
-----
`
        qc::round_up value trunc_places
    `

Description
-----------
Round up to the nearest $trunc_places decimal places

Examples
--------
```tcl

% qc::round_up 2.30 1
2.3
% qc::round_up 2.31 1
2.4
% qc::round_up 2.33333 2
2.34
```
