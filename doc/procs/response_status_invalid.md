qc::reponse status invalid
===========

part of [Connection Response API](../response_api.md)

Usage
-----
`qc::reponse status invalid`

Description
-----------
Set the status as "invalid" for the [connection response](../connection-response.md).

Examples
--------
```tcl

% qc::reponse status invalid
status invalid

```
