qc::frombase
============

part of [Docs](.)

Usage
-----
`
        qc::frombase base number
    `

Description
-----------
Converts a number from the specified base to base 10

Examples
--------
```tcl

% qc::frombase 16 F
15
% qc::frombase 2 1001101
77
% qc::frombase 8 77
63
```
