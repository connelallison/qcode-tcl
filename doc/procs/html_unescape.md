qc::html_unescape
=================

part of [Docs](.)

Usage
-----
`qc::html_unescape text`

Description
-----------
Convert html entities back to text

Examples
--------
```tcl

% qc::html_unescape [qc::html_escape {Hello <strong>Brave</strong> & "Wise" Ones}]
Hello <strong>Brave</strong> & "Wise" Ones

```
