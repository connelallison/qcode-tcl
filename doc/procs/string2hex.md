qc::string2hex
==============

part of [Docs](.)

Usage
-----
`qc::string2hex string`

Description
-----------
Convert string to hex

Examples
--------
```tcl

% string2hex Hello[format %c 256]
\x48\x65\x6c\x6c\x6f\xc4\x80
```
