qc::cast_boolean
================

part of [Casting Procs](../qc/wiki/CastPage)

Usage
-----
`qc::cast_boolean string ?true? ?false?`

Description
-----------
Cast a string as a boolean

Examples
--------
```tcl

% cast_boolean YES
t
%
% cast_boolean 0
f
%
% cast_boolean true Y N
Y

```
