qc::castable integer
==============

part of [Docs](../index.md)

Usage
-----
`qc::castable integer string`

Description
-----------
Test if the given string can be cast to an integer.

Examples
--------
```tcl

% qc::castable integer {1,234}
true
% qc::castable integer foo
false
```
