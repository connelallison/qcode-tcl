qc::date_year
=============

part of [Date Handling](../date.md)

Usage
-----
`qc::date_year date`

Description
-----------
Return the year.

Examples
--------
```tcl

% date_year 2007-05-06
% 2007
%
% date_year "last year"
% 2006

```
