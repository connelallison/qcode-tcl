qc::format_left
===============

part of [Docs](.)

Usage
-----
`qc::format_left string width`

Description
-----------
Padd with spaces left justified

Examples
--------
```tcl

% puts &quot;&gt;&gt;[format_left Dunstable 10]&lt;&lt;&quot;
&gt;&gt;Dunstable &lt;&lt;
% format_left Dunstable 6
Dunsta

```
