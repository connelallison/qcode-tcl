qc::format_left
===============

part of [Docs](.)

Usage
-----
`qc::format_left string width`

Description
-----------
Padd with spaces left justified

Examples
--------
```tcl

% puts ">>[format_left Dunstable 10]<<"
>>Dunstable <<
% format_left Dunstable 6
Dunsta

```
