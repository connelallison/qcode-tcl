qc::db_column_table
===========

part of [Database API](../db.md)

Usage
-----
`db_column_table column`

Description
-----------
Returns a list of tables that the given column name is part of.

Examples
--------
```tcl

% db_column_table user_id
user_perm file users session form comments

```
