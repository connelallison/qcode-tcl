qc::status get
===========

part of [Global JSON Response API](../response_api.md)

Usage
-----
`qc::status get`

Description
-----------
Get the status for the JSON Response.

Examples
--------
```tcl

% qc::status valid
status valid

% qc::status get
valid

%qc::status invalid
status invalid

%qc::status get
invalid

```
