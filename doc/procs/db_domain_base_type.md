qc::db_domain_base_type
===========

part of [Database API](../db.md)

Usage
-----
`db_domain_base_type domain_name`

Description
-----------
Returns the base type of the given domain.

Examples
--------
```tcl

% db_domain_base_type cardinal_number
int4

```
