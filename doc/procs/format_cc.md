qc::format_cc
=============

part of [Docs](.)

Usage
-----
`qc::format_cc cc_no`

Description
-----------
Format a credit card number in groups of 4 digits<br/>strip non digits

Examples
--------
```tcl

% format_cc 4111111111111111
4111 1111 1111 1111
% format_cc &quot;4111 1111 1111 1111&quot;
4111 1111 1111 1111

```
