qc::format_cc
=============

part of [Docs](../index.md)

Usage
-----
`qc::format_cc cc_no`

Description
-----------
Format a credit card number in groups of 4 digits<br/>strip non digits

Examples
--------
```tcl

% format_cc 4111111111111111
4111 1111 1111 1111
% format_cc "4111 1111 1111 1111"
4111 1111 1111 1111

```
