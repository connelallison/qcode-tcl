qc::widget_span
===============

part of [Docs](.)

Usage
-----
`
	widget_span name widgetName value text ?width size? ?height size?
    `

Description
-----------
Return a span element showing the value of the widget.

Examples
--------
```tcl

% widget_span name foo value bar
&lt;span id=&quot;foo&quot;&gt;bar&lt;/span&gt;

```
