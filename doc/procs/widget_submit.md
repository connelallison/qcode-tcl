qc::widget_submit
=================

part of [Docs](../index.md)

Usage
-----
`
	widget_submit name widgetName value buttonText ?option value? ?..?
    `

Description
-----------
Return an HTML form, submit button

Examples
--------
```tcl

widget_submit name foo value "Submit"
<input id="foo" value="Submit" name="foo" type="submit">

```
