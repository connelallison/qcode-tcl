qc::db_qualified_table_column
===========

part of [Database API](../db.md)

Usage
-----
`db_qualified_table_column column`

Description
-----------
Returns a fully qualified table column as a list of a pair: {table column}.

Examples
--------
```tcl

% db_qualified_table_column email
users email

```
