qc::call_with
===========

part of [Docs](../index.md)

Usage
-----
`call_with proc_name args`

Description
-----------
Calls proc_name by mapping name value pairs to named args.

Examples
--------
```tcl

% call_with qc::escapeHTML html "<p>Foo</p>"
&lt;p&gt;Foo&lt;/p&gt;

```
