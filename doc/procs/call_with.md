qc::call_with
===========

part of [Docs](../index.md)

Usage
-----
`call_with proc_name args`

Description
-----------
Calls the given proc with the arguments specified (as name value pairs).
Returns the result of the execution of the proc.

Examples
--------
```tcl

% call_with qc::escapeHTML html "<p>Foo</p>"
&lt;p&gt;Foo&lt;/p&gt;

```
