qc::is decimal
==============

part of [Docs](../index.md)

Usage
-----
`qc::is decimal int`

Description
-----------
Checks if the given string is a decimal number.

Examples
--------
```tcl

% qc::is decimal 1234
1
% qc::is decimal foo
0
% qc::is decimal 1.234
1
```
