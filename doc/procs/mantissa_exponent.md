qc::mantissa_exponent
=====================

part of [Docs](../index.md)

Usage
-----
`
        qc::mantissa_exponent x
    `

Description
-----------
Returns the mantissa and exponent which represent x

Examples
--------
```tcl

% qc::mantissa_exponent -0.000015463
-1.5463000000000002 -5
% qc::mantissa_exponent 912000000
9.120000000000001 8
```
