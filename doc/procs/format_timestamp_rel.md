qc::format_timestamp_rel
========================

part of [Docs](.)

Usage
-----
`qc::format_timestamp_rel string`

Description
-----------
Format relative to age with date and time

Examples
--------
```tcl

% format_timestamp_rel now
17:33
%
% format_timestamp_rel yesterday
Sun 17:34
%
% format_timestamp_rel &quot;next week&quot;
Sep 7th

```
