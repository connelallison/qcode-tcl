qc::response action resubmit
===========

part of [Connection Response API](../response_api.md)

Usage
-----
`qc::response action resubmit`

Description
-----------
Sets the resubmit element of the action object in the [connection response](../connection-response.md).

Examples
--------
```tcl

% qc::response action resubmit
action {resubmit {value true}}

```
