qc::cookie_exists
=================

part of [Cookie Handling](../cookie.md)

Usage
-----
`qc::cookie_exists name`

Description
-----------
Test if the cookie exists

Examples
--------
```tcl

% cookie_exists session_id
true
%
% cookie_exists foo
false

```
