qc::db_sequence_exists
===========

part of [Database API](../db.md)

Usage
-----
`db_sequence_exists ?-no-cache? sequence_name`

Description
-----------
Checks if the given sequence exists in the database.

Examples
--------
```tcl

% db_sequence_exists user_id_seq
true

```
