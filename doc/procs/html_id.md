qc::html_id
===========

part of [Docs](../index.md)

Usage
-----
`qc::html_id name ?value?`

Description
-----------
Wrap value in span tag and give it an ID

Examples
--------
```tcl

% html_id total 23.50
<span id="total">23.50</span>


```
