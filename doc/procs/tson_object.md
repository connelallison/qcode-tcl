qc::tson_object
===============

part of [Docs](.)

Usage
-----
`qc::tson_object args`

Description
-----------
Return a tson object from list of name value pairs in args

Examples
--------
```tcl

% qc::tson_object legs 4 eyes 2 coat fur call meow
object legs {number 4} eyes {number 2} coat {string fur} call {string meow}

```
