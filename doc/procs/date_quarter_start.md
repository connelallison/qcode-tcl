qc::date_quarter_start
======================

part of [Date Handling](../qc/wiki/DateHandling)

Usage
-----
`qc::date_quarter_start date`

Description
-----------
Return the date at the beginning of the quarter

Examples
--------
```tcl

% date_quarter_start 2007-08-06
% 2007-07-01

```
