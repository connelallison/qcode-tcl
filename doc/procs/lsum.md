qc::lsum
========

part of [Docs](.)

Usage
-----
`
        qc::lsum list
    `

Description
-----------
Returns the sum of a list of numeric items.

Examples
--------
```tcl

% set items [list 1 2 1 100]
1 2 1 100
% qc::lsum $items
104
```
