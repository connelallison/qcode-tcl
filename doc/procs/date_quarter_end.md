qc::date_quarter_end
====================

part of [Date Handling](../qc/wiki/DateHandling)

Usage
-----
`qc::date_quarter_end date`

Description
-----------
Return the date at the end of the quarter

Examples
--------
```tcl

% date_quarter_end 2007-08-06
% 2007-09-30

```
