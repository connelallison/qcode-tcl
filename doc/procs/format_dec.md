qc::format_dec
==============

part of [Docs](.)

Usage
-----
`qc::format_dec value dec_places`

Description
-----------
Round and commify decimal<br/>2314.235 -> 2,314.24

Examples
--------
```tcl

% format_dec 2314 3
2,314.000
%
% format_dec 865425833.2354 3
865,425,833.235

```
