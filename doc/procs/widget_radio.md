qc::widget_radio
================

part of [Docs](../index.md)

Usage
-----
`
	widget_radio name widgetName value checkedValue ?option value? ?..?
    `

Description
-----------
Return an HTML form, radio button

Examples
--------
```tcl

widget_radio name sex value male
<input id="sex" value="male" name="sex" type="radio">

```
